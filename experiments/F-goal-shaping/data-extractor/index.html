üöÄ Quick Start

Authentication

All API requests require authentication using an API key. Include your key in the request header:

curl -X POST https://api.extractorpro.com/v1/extract \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com", "extractors": ["text", "links", "images"]}'

Basic Extraction Example

// JavaScript Example
const response = await fetch('https://api.extractorpro.com/v1/extract', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    url: 'https://news.example.com',
    extractors: ['headline', 'author', 'date', 'content']
  })
});

const data = await response.json();
console.log(data);

Custom CSS Selectors

Use CSS selectors to target specific elements for extraction:

{
  "url": "https://example.com/products",
  "selectors": {
    "product_name": ".product-title h2",
    "price": ".price-tag span.amount",
    "rating": "div.stars[data-rating]",
    "description": "div.product-desc p"
  }
}

JavaScript Rendering

Extract content from JavaScript-heavy single-page applications:

{
  "url": "https://spa-example.com",
  "render_js": true,
  "wait_for": "#dynamic-content",
  "timeout": 5000
}

Pagination Handling

Automatically follow pagination links and extract from multiple pages:

{
  "url": "https://example.com/articles",
  "pagination": {
    "next_selector": "a.next-page",
    "max_pages": 10,
    "delay": 1000
  }
}
